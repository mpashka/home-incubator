# Рабочие допущения

- MVP запускается локально для доверенных пользователей без полноценной аутентификации.
- Поддерживаемые языки на старте: сербский (`sr`) и английский (`en`).
- Книга импортируется как один крупный текст (`book_texts`), а не как множество мелких строк-фрагментов; сложный parser форматов не входит в первый этап.
- Единица чтения (блок) не хранится в БД, а собирается динамически при чтении: позиция — символьное смещение, блок ограничен min/max размером и по возможности завершается на границе предложения/абзаца. Размер блока — свойство связки «пользователь + язык» (`user_language_skills`), растёт с уровнем владения. Обоснование и варианты — [`adr/0001-reading-blocks-and-text-storage.md`](adr/0001-reading-blocks-and-text-storage.md).
- Модель сложности слов (оценка known/unknown и целевая доля незнакомых слов в блоке) — будущее направление, не входит в первый vertical slice.
- Оценивание свободных ответов моделью и голосовой интерфейс не входят в первый backend vertical slice.
- Напоминания находятся вне scope текущего проекта.
