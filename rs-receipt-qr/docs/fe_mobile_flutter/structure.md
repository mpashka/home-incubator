# Структура файлов fe_mobile_flutter

- `lib/`
  - `main.dart` — Основная точка входа, домашняя страница, интеграция UI для сканирования QR и синхронизации.
  - `qr_scanner_page.dart` — UI сканера QR кода, логика для разбора QR, получения JSON счета и сохранения в Drift БД.
  - `qr_scanner_invoice_models.dart` — Объекты доступа к данным для счета и позиций, разобранных из JSON.
  - `local_db.dart` — Определение базы данных Drift (таблицы для чеков, позиций покупок и т.д.).
  - `sync_service.dart` — Логика синхронизации локальных чеков и покупок с бэкендом.
- `pubspec.yaml` — Зависимости проекта (Flutter, Drift, HTTP, сканер QR кодов и т.д.).
- Файлы, сгенерированные `build_runner` (например, `local_db.g.dart`) — Сгенерированы Drift для доступа к базе данных.

Другие стандартные каталоги Flutter (android/, ios/, web/ и т.д.) присутствуют для поддержки платформ.
