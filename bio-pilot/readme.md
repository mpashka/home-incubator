Приложение для улучшения здоровья и качества жизни.
Общая задача приложения - собирать информацию о здоровье и состоянии человека с использованием различных
устройств и приложений:
- фитнес трекер - сон, пульс, примерная оценка давления
- тонометр - давление
- весы - вес, баланс воды, висцериальный жир
- приложение на ПК - активность
- мобильное приложение для измерения калорийности и объёма еды и приёма пищи
- обычное расписание - например тренировки, график работы. по трекеру и по активности ПК можно отслеживать 
  время прихода/ухода с работы

Пользователь ставит цели. Например снижение веса, нормализация сна, сон с 22 до 7,
время работы с 8 до 17, повышение продуктивности.

Приложение собирает активность, даёт рекомендации. Например будильник. Напоминание. Рекомендация
лечь спать если активность пользователя высокая позже рекомендованного периода. Рекомендация
закончить работу и т.д. Рекомендация по питанию.
